Office 2016 customization tool silent install and integrating updates

2024-03-09
  1. setup /admin
  2. change -> Licensing and user interface (check I accept the terms in the License Agreement, choose Display level -> Basic, check Completion notice) -> Modify Setup properties (Add -> SETUP_REBOOT - Never) -> Set feature installation states (uncheck you dont want to install) -> Configure shortcuts (Remove you dont want to install)
  3. choose File -> Save as run.MSP file
  4. create a install bath with
@echo off

cd /d %~dp0

echo.
echo Installing Office, please wait...
echo.

setup.exe /adminfile run.MSP

exit

Download the Update for Microsoft Office 2016 32-Bit Edition from here.

https://learn.microsoft.com/en-us/officeupdates/office-updates-msi#office-2016-updates

To extract the .msp patches from the Microsoft Self-Extractor file (msodll40ui2016-kb5002469-fullfile-x86-glb.exe in this example), run the .exe file and use the /extract switch. For example, type the following at the command prompt:

msodll40ui2016-kb5002469-fullfile-x86-glb.exe /extract:C:\ExtractUpdates

Then copy all the Windows Installer patch (.msp) files in that folder to the Office Updates folder.
