rc.local is not running at boot

2024-03-06

  1. Ensure rc.local file is executable and make a testing

    sudo chown root /etc/rc.local
sudo chmod 755 /etc/rc.local
sudo bash /etc/rc.local

  2. Ensure rc-local.service enabled

    sudo systemctl enable rc-local.service

  3. if still no works, replace shebang line #!/bin/sh -e to #!/bin/bash then reboot
