Refresh Windows Icon Cache Without Rebooting

2024-03-09

https://superuser.com/questions/499078/refresh-icon-cache-without-rebooting

  1. You can just run the following command to clear the icon cache:
ie4uinit.exe -ClearIconCache
::for Windows 10
ie4uinit.exe -show
  1. Create a .bat file and paste the following lines in it and then run it.
taskkill /f /im explorer.exe
cd /d %userprofile%\AppData\Local
del IconCache.db /a
start explorer.exe
  1. If you don’t want to make a .bat, copy the line behind, press Win+R, paste it and press Enter.
cmd /c taskkill /f /im explorer.exe & del /a %userprofile%\AppData\Local\IconCache.db & start explorer

You can also paste it in the address bar of an explorer window; and, of course, in a command line window (you don’t need the cmd /c then) but you won’t have the line to easily repeat it in case you need it again.
