Install Sphinx

First you need to install Python on your windows.

When you finished that,you can install sphinx use this command: :

pip install sphinx

Untill it is done.Then you can exec sphinx-quickstart to build the sphinx template for quick starting.

Modify index.rst:

.. toctree::
  :maxdepth: 1
  :caption: Contents:
    
  chapter1
  chapter2
  ...

Then you can exec command as below to verify it.

sphinx-build -b html sourcedir builddir [make html]
  • http://www.sphinx-doc.org/en/stable/tutorial.html

Install Theme

Exec command as below:

pip install sphinx_rtd_theme

Then modify conf.py:

"html_theme = "sphinx_rtd_theme" 
"html_theme_options"
...

and build again: :

sphinx-build -b html source build/html [make html]
  • https://github.com/rtfd/sphinx_rtd_theme/
  • https://github.com/guzzle/guzzle_sphinx_theme

To make the ReadTheDocs theme use the entire width of your screen you can modify the theme.css file, removing the max-width: 800px; property from the wy-nav-content class definition, like so:

.wy-nav-content {
  padding: 1.618em 3.236em;
  height: 100%;
  /* max-width: 800px; */
  margin: auto;
}

Thanks to chown !

  • https://stackoverflow.com/questions/23211695/modifying-content-width-of-the-sphinx-theme-read-the-docs

Install rst2pdf

Exec command as below: :

pip install rst2pdf [easy_install rst2pdf]

Then modify conf.py: :

extensions=['rst2pdf.pdfbuilder']
pdf_documents = [(u'index', u'pdf file name', u'pdf title', u'author name'),]
...
  • http://rst2pdf.ralsina.me/handbook.html

And Build again. :

sphinx-build -b pdf source build/pdf [make pdf]
